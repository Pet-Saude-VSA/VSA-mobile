# 📱 Pet-Saúde VSA - Mobile

Aplicativo móvel desenvolvido com React Native e Expo, focado em funcionamento Offline-First utilizando SQLite.

## 🚀 Como rodar o projeto localmente

1. **Instale as dependências:** (antes: apague o node modules e o package-lock.json)
   `npm install`

2. **Inicie o servidor do Expo:**
   `npx expo start`

3. **Teste no seu telemóvel:**
   Baixe a aplicação "Expo Go" no seu smartphone e faça scan do QR Code que aparecerá no terminal.# 📱 Pet-Saúde VSA - Mobile
